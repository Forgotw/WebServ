# stressTest

## How to use

```text
./stressTest -h
Usage of ./stressTest:
  -duration int
        Duration of the test in seconds (default 30)
  -ip string
        IP to send requests to (default "localhost")
  -port string
        Port to send requests to (default "8080")
  -size int
        Size of the payload (default 0)
  -v    Show response of each request
```
